# Icons Directory

Place your custom icons here for the About section.

## Required Icons

Add the following icon files (SVG, PNG, or JPG format):

1. **web-development.svg** (or .png/.jpg) - For Web Development section
2. **data-analytics.svg** (or .png/.jpg) - For Data Analytics section  
3. **database.svg** (or .png/.jpg) - For Databases & Cloud section

## File Naming

The icons should be named exactly as:
- `web-development.svg` (or .png/.jpg)
- `data-analytics.svg` (or .png/.jpg)
- `database.svg` (or .png/.jpg)

## Fallback

If these local icons are not found, the site will automatically use CDN icons from Simple Icons as a fallback.

## Recommended Size

Icons should be approximately 64x64 pixels for best display quality.
